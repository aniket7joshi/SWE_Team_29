About

The application aims for the detection of plagiarism in handwritten or typed documents like reports, essays, written assignments, etc. using NLP for assessing similarity and CV for handwriting recognition. The application checks for plagiarism on several levels:

Exact matches between pairs, creating clusters

Checks for exactly matching strings in uploaded documents

Fuzzy matches between pairs

Approximate matching between documents, i.e having high similarity

Synonym matches using WordNet

Matching between document strings using synonyms of words

Experimental text similarity using pre-trained NLP models

Using pre-trained models to find similar documents
